# 🎉 持仓监控分时图 - 快速开始

## ✨ 新功能上线！

持仓监控现在支持**专业的股票分时图**，就像同花顺一样！

---

## 🚀 3步快速体验

### 第1步：打开持仓监控
```
运行 Streamlit 应用 → 点击 "📊 持仓监控"
```

### 第2步：展开股票信息
点击任意股票的展开区域，例如：
```
📊 平安银行 (000001) - 当前价: 11.63 | 涨跌幅: +0.69%
```

### 第3步：查看分时图
在操作区域点击 **"📈 分时图"** 按钮，分时图立即展开！

---

## 🎨 分时图特点

### 专业配色（类似同花顺）
- 🌙 **深蓝色背景** - 专业舒适
- ⚪ **白色价格线** - 清晰醒目
- 💛 **黄色均价线** - 经典配色
- 🔴🟢 **红涨绿跌** - 符合习惯

### 丰富信息
- 📈 **实时价格走势** - 1分钟级别
- 📊 **均价线** - 自动计算当日均价
- 📉 **昨收价参考线** - 方便对比涨跌
- 💹 **成交量柱状图** - 红涨绿跌

### 详细统计
- 最高价 | 最低价 | 振幅 | 成交量
- 数据更新时间

## 💡 使用技巧

### 技巧1：判断强弱
- 价格在均价线**上方** → 强势 💪
- 价格在均价线**下方** → 弱势 📉
- 价格围绕均价**波动** → 震荡 〰️

### 技巧2：观察成交量
- **放量上涨** → 买盘积极 🔥
- **放量下跌** → 卖盘积极 ❄️
- **缩量** → 观望情绪 😴

### 技巧3：关注昨收价
- 价格在昨收价**上方** → 多头占优 🐂
- 价格在昨收价**下方** → 空头占优 🐻
- 反复测试昨收价 → 关键位置 ⚡

---

## ⚠️ 注意事项

1. **交易时间限制**
   - 分时数据仅在交易时间可用
   - 交易时间：周一至周五 9:30-11:30, 13:00-15:00

2. **数据延迟**
   - 数据可能有1-3分钟延迟
   - 仅供参考，不建议用于实盘交易

3. **首次加载**
   - 需要加载240条1分钟数据
   - 首次加载可能需要2-3秒
